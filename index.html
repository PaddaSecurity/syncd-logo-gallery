The Logo Gallery

V1 Usage

The V1 logo gallery offers a collection of static logos in both SVG and PNG formats. These pre-rendered image files are ready for direct embedding or linking.

To use a V1 logo from the gallery, use one of the following URL formats:

For SVG:

https://syncdpublic.dev/logos/logo.svg

For PNG:

https://syncdpublic.dev/logos/logo.png

These URLs will return only the logo file, without any additional markup or wrappers.

V2 Usage

The V2 folder (/logos-v2) contains a different set of logos with a new naming convention. V2 logos are available only in SVG format.

To use a V2 logo from the gallery, use the following URL format:

https://syncdpublic.dev/logos-v2/logo.svg

For logos with dark and light themes:

https://syncdpublic.dev/logos-v2/logo.dark.svg
https://syncdpublic.dev/logos-v2/logo.light.svg

Note: Some V2 logos are only available in one theme. In that case, access them using the standard format: 

https://syncdpublic.dev/logos-v2/logo.svg

Available Logos

To see all available logos and their variations, please visit our GitHub repository: 

https://github.com/PaddaSecurity/syncd-logo-gallery

In the repository, you'll find:

  • A complete list of all V1 and V2 logos
  • Preview images of each logo
  • Information about available formats and themes for each logo

You can also contribute to the project by submitting new logos or suggesting improvements through GitHub issues or pull requests.

Logo Takedown Requests

If you believe that a logo in our gallery infringes on your intellectual property rights and you want it removed, please email your takedown request to: 

tcistull@syncd.dev

We will review your request and take appropriate action as soon as possible.
